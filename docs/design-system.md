# Design System

The current design follows the Figma final section `409:11104`.

## Colors

- Background: `#FCF8F9`
- Ink / Primary: `#393C43`
- Surface: `#FFFFFF`
- Navigation surface: `#F0EDED`
- Border: `#C6C6CB`
- Muted text: `#8A8D96`

## UI Rules

- Use `AppColors` and `AppTheme`.
- Prefer restrained cards, clean sections, and 12-18px radii.
- Use Lucide icons only.
- Replace Figma `Curated` copy with `PopupMate`.
- Use local gradients as image placeholders until final assets are exported.
